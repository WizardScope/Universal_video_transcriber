# Universal_video_transcriber — русская инструкция

Универсальный Python-скрипт для транскрибации аудио и видео на базе `faster-whisper` с постобработкой текста, защитой от зацикливания декодера и автоматической генерацией итоговых материалов.

## Что умеет

- транскрибировать аудио и видео;
- работать с разными типами контента: лекции, подкасты, созвоны, интервью, видеоуроки;
- очищать и нормализовать текст после распознавания;
- отсеивать служебные и слабые сегменты;
- защищаться от повторяющихся зацикленных фрагментов;
- формировать несколько итоговых файлов.

## Что создаёт на выходе

По умолчанию скрипт сохраняет:

- `*_full_readable.txt` — полный читаемый текст;
- `*_brief.txt` — краткое содержание по разделам;
- `*_study_pack.txt` — вопросы, карточки, словарь терминов и план повторения.

## Требования

Перед запуском должны быть установлены:

- Python 3.10+
- `ffmpeg` и `ffprobe` в `PATH`
- Python-библиотеки из `requirements.txt`

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone https://github.com/WizardScope/Universal_video_transcriber.git
cd Universal_video_transcriber
```

### 2. Создать и активировать виртуальное окружение

#### Windows
```bash
python -m venv .venv
.venv\Scripts\activate
```

#### Linux / macOS
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Проверить FFmpeg

```bash
ffmpeg -version
ffprobe -version
```

Если команды не находятся, нужно установить FFmpeg и добавить его в `PATH`.

### 5. Указать путь к входному файлу и папке для результата

Откройте файл `Universal_video_transcriber_v3_6_1.py` и заполните в блоке настроек:

```python
INPUT_MEDIA = r"путь_к_вашему_файлу"
OUTPUT_DIR = r"папка_для_результата"
BASE_NAME = ""
```

### 6. При необходимости изменить режим работы

В этом же блоке можно поменять:

```python
DEVICE_MODE = "cuda"      # "cuda" / "cpu" / "auto"
PROFILE = "quality"       # "fast" / "balanced" / "quality"
CONTENT_TYPE = "auto"     # "auto" / "lecture" / "meeting" / "podcast" / "generic"
LANGUAGE = "ru"           # None = автоопределение языка
```

Если нет GPU или CUDA не настроена, поставьте:

```python
DEVICE_MODE = "cpu"
```

### 7. Запустить скрипт

```bash
python Universal_video_transcriber_v3_6_1.py
```

## Пример результата

После завершения работы в папке `OUTPUT_DIR` появятся файлы вида:

```text
example_full_readable.txt
example_brief.txt
example_study_pack.txt
```

Дополнительно могут создаваться технические файлы, если включить соответствующие флаги в настройках.

## Полезные замечания

- Скрипт сейчас запускается через редактирование блока `USER SETTINGS` внутри файла.
- Для работы в офлайн-режиме используется локальный кэш моделей.
- Если GPU недоступен или не подходит, скрипт умеет переключаться на CPU.
- Для длинных файлов есть промежуточные сохранения результата.
